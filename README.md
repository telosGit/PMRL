# PoC MALWARE RESEARCH LICENSE (PMRL)
A license for Proof-of-Concept/Malware projects


Why does this exist?
---

For proof-of-concept malware that is meant to exibit the dangers of a vulnerability, but not intended/designed for actual harmful use. For example [tarball](https://github.com/telosGit/tarball) uses this.

How do I use it?
---

1. Copy the contents of [LICENSE-template](https://github.com/telosGit/PMRL/blob/main/LICENSE-template)

2. Paste it into a new file in the root (`/`) of your repo called `LICENSE` (no extention)

4. Replace the `[year]`, `[name/handle/username]`, and `[contact]` blocks in the license with your correct info. (on lines `4`, `65`, `141`)

3. Save, reference this repository in the `README.md` of your repo please.
